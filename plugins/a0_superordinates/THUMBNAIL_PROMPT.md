# A0 Superordinates Plugin - Thumbnail Generation Prompt for Nanobanana

## Visual Concept
Create a **square (1:1 aspect ratio) thumbnail image** that represents the a0_superordinates plugin's core functionality: hierarchical agent management, persistent subordinate agents, and organizational structure.

## Key Visual Elements

### Primary Symbolism
- **Hierarchical Tree Structure**: Show a tree or nested boxes representing parent-child agent relationships
- **Agents/Nodes**: Use circular nodes or avatar-like icons to represent individual agents
- **Organization**: Convey a sense of structure, organization, and team collaboration
- **Connection Lines**: Show connections between parent and subordinate agents

### Color Scheme
- **Primary Color**: Deep blue or teal (professional, technical)
- **Accent Colors**: Purple or cyan for highlights and connections
- **Background**: Clean, minimal (white, light gray, or subtle gradient)
- **Nodes/Icons**: Use contrasting colors (orange, green, or gold) for individual agents

### Composition
- **Center Focus**: Place the main hierarchical structure in the center
- **Top-Level Agent**: Show a primary/root agent at the top or center
- **Subordinate Agents**: Display 2-4 child agents branching downward or outward
- **Connections**: Clean lines or subtle glow effects connecting agents
- **Spacing**: Leave adequate padding around edges (10-15% margin)

## Specific Design Suggestions

### Option 1: Tree Structure
- Root circle/node at top center
- Three child nodes below at different levels
- Clean, minimalist connecting lines
- Simple geometric style
- Total: 5-6 visible nodes

### Option 2: Organizational Chart
- Central agent (primary)
- Surrounding secondary agents in circular arrangement
- Nested or tiered layout
- Connection points between entities
- Modern, clean typography with agent names ("A0", "Dev", "Res", etc.) optional

### Option 3: Network/Mesh
- Interconnected nodes showing bidirectional communication
- Emphasize the persistent, always-available nature of subordinates
- Use subtle animation-like arrows or flow indicators
- Modern, technical aesthetic

## Technical Constraints
- **Aspect Ratio**: Exactly 1:1 (square)
- **File Size**: ≤ 20 KB (use compression)
- **Format**: PNG, JPG, JPEG, or WEBP
- **Resolution**: Recommend 256x256 or 512x512 pixels
- **Style**: Modern, flat or minimal 3D design preferred over photorealistic

## Tone & Style
- **Professional**: Suitable for a technical tools marketplace
- **Modern**: Clean, contemporary design language
- **Friendly**: Approachable, not intimidating
- **Clear**: Easy to recognize at small sizes (favicon-like clarity)

## Do's
✅ Use geometric shapes (circles, boxes, lines)
✅ Employ a limited color palette (3-5 colors max)
✅ Show hierarchical or organizational relationships clearly
✅ Keep text minimal or absent (symbols communicate better at thumbnail size)
✅ Use consistent line weights and spacing
✅ Design with dark and light backgrounds in mind

## Don'ts
❌ Avoid photorealistic elements or complex gradients
❌ Don't use small text or fine details (won't render at small sizes)
❌ Avoid overly busy designs with too many elements
❌ Don't use colors that clash or feel unprofessional
❌ Avoid 3D rendering styles that are overly complex

## Use Case Context
This thumbnail will be displayed in:
- The a0-plugins GitHub repository Plugin Index
- The Agent Zero Plugin Hub (Plugins dialog → Browse tab)
- Plugin listing cards (typically 128-200px)
- Search results within the Plugin marketplace

The image should be **immediately recognizable** and **professional-looking** even at small sizes.
